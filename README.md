## System Requirements

Admin App - Web

`Run properly on web browser`

Client App - Android

`Gradle: 6.7`

`minSdkVersion: 21`

`targetSdkVersion: 30`

## Technology

**NodeJs - ReactJs - Flutter - Dialogflow**
